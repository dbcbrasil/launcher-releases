# DBC Brasil Launcher

Downloads oficiais do launcher do servidor DBC Brasil.

## Baixar

Abra a aba **Releases** e escolha o instalador do seu sistema:

- Windows: `.exe` ou `.msi`
- Fedora: `.rpm`
- Ubuntu/Debian: `.deb`
- macOS: `.dmg`

## Suporte

Se tiver problemas, entre no Discord oficial da DBC Brasil.

## Aviso

Este repositorio contem apenas os instaladores publicos. O codigo-fonte do launcher nao e publicado aqui.
